# 🚀 AI Marketing Agent (DevOps Project)

## 📌 Overview

This project is an AI-powered Marketing Agent that generates marketing content using Groq API. It demonstrates a complete DevOps workflow including Docker containerization, DockerHub integration, Kubernetes orchestration, and deployment on AWS EC2.

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Groq API
* Docker
* DockerHub
* Kubernetes (Minikube)
* AWS EC2

---

## 🚀 Features

* AI-based content generation
* Web-based UI using Streamlit
* Dockerized application
* Cloud deployment on AWS
* Kubernetes support for orchestration

---

## 🐳 Run using Docker

```bash
docker run -d -p 8000:8000 -e GROQ_API_KEY=your_api_key dollynasahu26/marketing-agent
```

---

## ☁️ Deployment

Application is deployed on AWS EC2 and can be accessed via:

```
http://15.207.14.168:8000/
```
